The AI interview simulator is a multimodel interview preparation system that combines natural language processing, sentiment polarity and emotion recognition to enhance the interview skills of the user. The main purpose of this project is to help it's users in preparing for their interviews and increase their chances of employability.
It also provides novel features like top answer suggestion module and an adaptive learning mechanism based on analyzing user responses. The aim of this project is to develop an emotion aware system that acts like a real coach and gives actual good advice and learns from user responses to give better feedback each time, allowing growth and better user experience compared to generic feedback systems and adaptive questionning.
